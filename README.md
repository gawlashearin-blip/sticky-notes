# 桌面便签 (Sticky Notes)

一个轻量的 Windows 桌面便签程序，可以在桌面任意位置贴便签、写字、调整大小。基于 WPF + .NET 8 开发，常驻内存约 30–60 MB。

## 特性

- 在桌面任意位置创建多张便签，重叠时由系统自然遮挡
- 富文本编辑，支持加粗 / 斜体 / 下划线
- 自由拖拽调整大小（带最小尺寸限制）
- 便签置顶（钉住）、折叠、六种颜色切换
- 内容、位置、大小、颜色、状态自动保存，关机重开原样恢复
- 支持开机自启
- 单文件打包，双击即用，无需安装运行时

## 安装与运行

### 直接使用（推荐）

1. 前往 [Releases](../../releases) 下载打包好的压缩包
2. 解压到任意文件夹
3. 双击 `StickyNotes.exe` 即可运行

> 首次运行时 Windows SmartScreen 可能提示未知发布者，点击「更多信息」→「仍要运行」即可。
> 解压后请保持文件夹内所有文件在一起，不要单独移动 exe。

### 从源码构建

需要安装 [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)。

```bash
# 克隆仓库
git clone https://github.com/gawlashearin-blip/sticky-notes.git
cd sticky-notes

# 运行
dotnet run

# 打包成自包含单文件 exe（不依赖系统 .NET）
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o publish/self-contained
```

打包产物在 `publish/self-contained/` 目录下。

## 使用说明

### 新建便签
- 左键单击右下角托盘图标，新建一张便签
- 或右键托盘图标 →「新建便签」
- 每张新便签会自动错开位置，不会完全重叠

### 写字 / 富文本
- 直接在便签正文区打字，支持中英文、回车换行
- 选中文字后使用快捷键设置格式：`Ctrl+B` 加粗、`Ctrl+I` 斜体、`Ctrl+U` 下划线
- 这些快捷键是开关，有格式时再按一次即可取消
- 打字会延续光标当前格式，想打普通字时先按对应快捷键关掉格式（与 Word 一致）

### 移动便签
- 按住顶部标题条中间区域拖动，即可移动整张便签

### 改变大小
- 把鼠标移到便签四条边或四个角，光标变成箭头后拖动即可缩放
- 有最小尺寸限制（150 × 120），缩不到更小

### 标题条按钮（从左到右）
- 📌 **钉住** — 点击让便签置顶，浮在所有程序（浏览器、Word 等）之上；再点取消
- ─ **折叠** — 点击只保留标题条窄条，节省空间；再点恢复，内容不丢
- 🎨 **颜色** — 点击弹出六种颜色（黄 / 绿 / 蓝 / 粉 / 紫 / 橙），每张可独立设色
- ✕ **删除** — 删掉这一张便签，不影响其他便签

### 开机自启
- 右键托盘图标 →「开机自启」，前面打钩即开启，再点取消
- 注意：开启后请勿移动程序所在文件夹，否则需在新位置重新打钩一次

### 退出程序
- 右键托盘图标 →「退出」关闭整个程序
- 删光所有便签不会退出，托盘图标仍在，可继续新建

## 数据存储

所有便签数据保存在：

```
%AppData%\StickyNotes\notes.json
```
